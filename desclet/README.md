# My Desklet

Простой десклет для Linux Mint (Cinnamon).

## Установка

1. Скопируйте папку `desclet` в `~/.local/share/cinnamon/desklets/`
2. Переименуйте папку в `my-desklet@example` (или другое имя, соответствующее UUID в metadata.json)
3. Перезапустите Cinnamon (Alt+F2, введите `r`)

## Использование

1. Правый клик на рабочем столе → Добавить десклеты
2. Найдите "My Desklet" в списке
3. Добавьте на рабочий стол

## Разработка

- `metadata.json` - метаданные десклета
- `desklet.js` - основной код
- `stylesheet.css` - стили
